安装插件

当前是内测安装包，请在电脑浏览器里下载 zip，解压后手动加载一次。iPad 和手机浏览器不能安装这个插件。正式上架 Edge Add-ons 或 Chrome Web Store 后，会改成商店一键安装。

下载 Edge 插件包 下载 Chrome 插件包
Edge 开发人员模式在哪里？ 打开 edge://extensions 后，看页面左侧栏下方，“开发人员模式”开关在那里；窗口较窄时可能在扩展页面的侧栏或菜单里。
Chrome 开发者模式在哪里？ 打开 chrome://extensions 后，看页面右上角，“开发者模式”开关在那里。
加载后去哪里操作？ 打开一条豆包聊天，在页面右上角找到“迁移当前对话”。如果没看到插件图标，可点击浏览器右上角的拼图图标，将“豆包迁移助手”固定到工具栏。
为什么之前只同步了后半段？ 豆包聊天页会虚拟滚动，页面里经常只保留当前附近的消息。新版插件点击同步时会自动向上滚动采集历史，再按豆包消息 ID 排序后上传。

图文安装教程

下面的图按真实操作入口画好了。Edge 可以从右上角三点菜单进入“管理扩展”，也可以直接在地址栏打开扩展页。

Edge：从菜单进入管理扩展

先点右上角三个点，选择“扩展”，再点“管理扩展”。进入扩展页后打开左侧栏下方“开发人员模式”。

  1. 下载 Edge 插件包，把 zip 解压到固定文件夹。
  2. 点右上角三个点，进入“扩展” → “管理扩展”；也可以直接打开 edge://extensions。
  3. 打开左侧栏下方“开发人员模式”。如果左侧栏收起，先点页面左上角菜单展开。
  4. 点击“加载解压缩的扩展”。
  5. 选择刚才解压出来的插件文件夹。

Chrome：右上角打开开发者模式

地址栏输入 chrome://extensions，打开右上角“开发者模式”，再点“加载已解压的扩展程序”。

  1. 下载 Chrome 插件包，把 zip 解压到固定文件夹。
  2. 打开 chrome://extensions。
  3. 打开右上角“开发者模式”。
  4. 点击“加载已解压的扩展程序”。
  5. 选择刚才解压出来的插件文件夹。

迁移流程图

这个流程不需要复制 Token，也不需要输入豆包账号密码。推荐先把当前豆包聊天完整保存到电脑，再由你主动上传到 2link。

第一步：在豆包完整导出到本地

进入要迁移的豆包聊天，在页面右上角点击“迁移当前对话”，扫描完成后优先选择“完整导出到本地”。

第二步：在 2link 上传本地迁移数据

打开 2link，找到“已同步的外部智能体”，点击“上传本地迁移数据”，选择刚才导出的文件夹。

1. 登录 2link

打开 2link，登录账号，并确认已经绑定购买过的授权码。

2. 连接账号

点击浏览器里的插件图标，再点“连接 2link 账号”。

3. 打开豆包聊天

在同一个浏览器里登录豆包，进入要迁移的聊天页面。

4. 完整导出到本地

点击“迁移当前对话”，优先选择完整导出，保存文字、图片和视频。

5. 上传到 2link

在 2link 的“已同步的外部智能体”区域点击“上传本地迁移数据”。

6. 少量文字可直接同步

迁移弹窗第三项是“直接同步到 2link”，仅适用于不超过 5,000 条且不超过 10 MB 的纯文字数据；超出后按钮会自动停用。

迁移成功后在哪里看？

聊天记录会先进入你的 2link 账号数据库。后续可以在 2link 里继续导出数据、做云备份、或恢复到自己的 2link 数据里。

可以迁移全部历史吗？

新版插件会在同步时自动向上滚动采集历史消息，并按豆包消息 ID 排序。极长对话如果豆包加载很慢，可以先手动向上滚动一遍，再点击同步。

常见问题

如果同步失败，先按下面几项排查。

提示需要登录或绑定授权码

请先打开 2link，登录账号，在账号与授权区域绑定已购买的授权码。绑定后回到插件里重新点击“连接 2link 账号”。

豆包页面没有“迁移当前对话”按钮

确认当前页面是豆包聊天详情页，并刷新一次页面。如果仍然没有出现，请检查扩展管理页里“豆包迁移助手”是否启用；浏览器工具栏没看到图标时，先点右上角拼图图标查看已安装扩展。

同步后消息数量太少

豆包网页通常只渲染当前已加载的内容。请先向上滚动加载更多历史消息，再点击同步。

隐私和账号安全

插件不会要求你输入豆包账号密码，也不会把豆包 Cookie 保存到 2link。同步时会把当前聊天页识别出的消息内容上传到 2link 服务器，用于写入你的账号数据库。
